\## Descripción

Este proyecto consiste en la personalización de una plantilla HTML obtenida en w3layouts.com. Incluye estructura web completa, contenido original, organización 

de recursos y despliegue mediante GitHub Pages.

\- HTML5  

\- CSS3  

\- JavaScript (aún no trabajado) 

\- Git \& GitHub  

\- GitHub Pages
